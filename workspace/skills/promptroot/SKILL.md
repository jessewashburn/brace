# PromptRoot Skill

PromptRoot is Jesse's versioned prompt library, hosted at `github.com/promptroot/promptroot`.

## Fetch a prompt by slug

```bash
bash workspace/skills/promptroot/scripts/fetch-prompt.sh promptroot/promptroot main <slug>
```

Slugs mirror the file path under `prompts/` with `.md` stripped:
- `tutorial/templates/versioned-modular-sdd-plan`
- `code-review`
- `onboarding/new-engineer`

If the slug isn't found on the specified branch, the script automatically falls back to `main` and notifies you.

### Variable substitution

Fetched prompts may contain `{PLACEHOLDER}` variables. After fetching, scan for them and ask Jesse for values before executing:

```bash
CONTENT=$(bash workspace/skills/promptroot/scripts/fetch-prompt.sh promptroot/promptroot main <slug>)
# Detect placeholders
echo "$CONTENT" | grep -oP '\{[A-Z_]+\}' | sort -u
# Prompt Jesse for each value, then substitute:
RENDERED="${CONTENT/\{MY_VAR\}/replacement}"
```

## List available prompts

```bash
bash workspace/skills/promptroot/scripts/list-prompts.sh promptroot/promptroot main
```

Returns one slug per line.

## Contribute a prompt back (Phase 2)

Only propose a PR when at least one trigger criterion is met:
1. Jesse explicitly says "save this as a prompt" or "add this to PromptRoot"
2. The same ad-hoc instruction pattern appeared ≥2 times in the last 7 days (check session memory)
3. The session produced a standalone artifact (SDD, test plan, PR description) that didn't use an existing PromptRoot template

**Always ask Jesse for confirmation before opening a PR.**

Contribution workflow:
```bash
# 1. Create a branch in promptroot/promptroot
gh api repos/promptroot/promptroot/git/refs \
  --method POST \
  -f ref="refs/heads/bliz/add-<slug>" \
  -f sha="$(gh api repos/promptroot/promptroot/git/ref/heads/main --jq '.object.sha')"

# 2. Create the file
gh api repos/promptroot/promptroot/contents/prompts/<slug>.md \
  --method PUT \
  -f message="Add <slug> prompt [bliz-generated]" \
  -f branch="bliz/add-<slug>" \
  -f content="$(base64 -w0 <<< "$PROMPT_CONTENT")"

# 3. Open the PR
gh pr create \
  --repo promptroot/promptroot \
  --head bliz/add-<slug> \
  --base main \
  --title "Add <slug> prompt" \
  --body "Generated by Bliz. Tags: [bliz-generated]"
```

Then send Jesse the PR URL via Telegram.

## Error handling

- **Rate limit:** scripts retry up to 3 times with exponential backoff, then exit 2. If this happens, notify Jesse via Telegram and skip the operation.
- **Slug not found:** falls back to `main` branch automatically, then exits 1 if still not found.
- **Network failure:** script exits non-zero with an error message — surface this to Jesse, do not silently skip.
